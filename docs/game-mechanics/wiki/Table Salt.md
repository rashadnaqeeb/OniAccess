# Table Salt

Table Salt
A seasoning that Duplicants can add to their
Food
to boost
Morale
.
Only the finest grains are chosen.
Type
Miscellaneous
Table Salt
is a Miscellaneous Resource refined from
Salt
.
Usage
Table Salt can be stored in
Mess Tables
or
Communal Tables
. Duplicants will consume Table Salt when eating at tables.
Each Mess Table can store up to 5 g Table Salt and each Communal Table can store up to 15 g of Table Salt. Each usage consumes 1 g Table Salt and grants +1 Morale bonus to Duplicants.
Production
The
Rock Crusher
can refine
Salt
into Table Salt. 100 kg Salt = 5 g Table Salt + 100 kg
Sand
.
Notes
Table Salt is not an element, instead its element is Salt.
Bionic Duplicants
will consume Table Salt when ingesting
Power Banks
at tables, same as Standard Duplicants, despite Power Banks not being
Food
