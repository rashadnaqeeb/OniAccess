Ceiling Trim
is a
decorative
building
for ceilings. It provides a small decor value of +5. Building it with
Sandstone
can provide a +10% decor bonus, and building it with
Granite
or
Ceramic
can provide a +20% decor bonus.
Tips
Ceiling Trims can be used to
order Dupes to dig a cell if it gets covered by a natural tile
, such as
Regolith
falling from
Meteor Shower
.
It can also be used to temporarily restrict
Pip
planting, without having to deconstruct a building like
Ladder
when the place needs to be emptied for Pip planting later.
