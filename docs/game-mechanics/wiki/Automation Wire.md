Automation wires
connect
automatable buildings
and
sensors
to create automated systems. The wire is green if any connected output is green and red otherwise. All endpoints of connected wires are instantly and simultaneously updated.
This wire exists on the
automation layer
, meaning it can cross through most other objects (like
Tiles
); to cross over itself, an
automation wire bridge
can be used.
Tips
Sending an
Automation Signal
with a higher
Bit Depth
than the connected
Logic Wire
can result in damage to the circuit. In other words,
automation ribbon
outputs where the 2nd, 3rd, or 4th bit is green will cause overloading damage to connected automation wires.
