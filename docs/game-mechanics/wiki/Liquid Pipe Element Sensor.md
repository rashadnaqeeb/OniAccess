Liquid Pipe Element Sensor
is an automation sensor that detects the content of a
Liquid Pipe
. It has an element filter for player to choose a type of Liquid. It must be built in the same tile as a Liquid Pipe segment. When the chosen Liquid is detected in the pipe segment, it will output a Green signal. Otherwise it will output a Red signal.
If the liquid pipe is empty, it outputs a Red signal.
Usage
Liquid Pipe Element Sensor combined with
Liquid Shutoff
works just as
Liquid Filter
, but consumes less power (10 W vs. 120 W).
