The
Remote Controller
is used in conjunction with the
Remote Worker Dock
to enable remote operations. The Remote Controller links directly to the Dock, allowing a Duplicant to operate various errands from a distance without needing to be physically present. The Remote Controller itself consumes 0.0133 units/s of
Data Bank
during operation and must periodically be resupplied. In addition, the
Remote Worker Dock
consumes 33.33g/s of
Gear Oil
during operation. Remote Controllers can only be connected to Docks located on the same asteroid.
For a full list of tasks that can be performed remotely once connected, see the
Remote Worker Dock
page.
