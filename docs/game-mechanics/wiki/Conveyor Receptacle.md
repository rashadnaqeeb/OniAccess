# Conveyor Receptacle

Conveyor Receptacle
Unloads Solid Materials from a Conveyor Rail into storage.
When materials reach the end of a rail they enter a receptacle to be used by Duplicants.

Conveyor Receptacle
serves as an output storage container for Solid Materials from a
Conveyor Rail
system.
Solid Materials can be removed from a Conveyor Receptacle using an
Auto-Sweeper
with a
Storage Bin
near by if the container has a higher priority than the receptacle.
Requires Duplicant with Mechatronics Engineering
skill
to build.
Tips
It stores only 100 kg (or 5 tiles of rail network) and therefore, probably, should be augmented with Auto-Sweeper and end-point Storage Bins nearby.
Check the box "Allow Manual Use" to allow Duplicants to access the storage.
Duplicants and Auto-Sweepers need to reach the bottom tile of the receptacle to access its contents.
Auto-Sweepers will not remove Solid Materials to load a
Conveyor Loader
that is a part of the same rail system. Separating the rail system such as through a
Conveyor Bridge
is required to load from a Conveyor Receptacle.
