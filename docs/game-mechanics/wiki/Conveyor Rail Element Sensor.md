Conveyor Rail Element Sensor
is an automation sensor that detects the content of a
Conveyor Rail
. It has an element filter for the player to choose a type of element. It must be built in the same tile as a Conveyor Rail segment. When the chosen element is detected in the rail, it will output a Green signal. Otherwise it will output a Red signal.
If the rail is empty, it outputs a Red signal.
Usage
Conveyor Rail Element Sensor combined with
Conveyor Shutoff
works just as
Solid Filter
, but consumes less power (10 W vs. 120 W).
