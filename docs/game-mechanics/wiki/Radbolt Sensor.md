Radbolt Sensor
sends a green automation signal or a red automation signal when a
Radbolt
is present, matching the conditions specified. If no Radbolt is present, the sensor may still return a green signal, depending on the settings (eg, "below 10 Radbolts" will return green until a Radbolt with a strength higher than 10 Radbolts is seen.)
A Radbolt must pass directly on top of the sensor for the sensor to detect the radbolt. Near misses will not register, and the radbolt is only present for a moment, so only a single pulse will be sent by the sensor as the radbolt crosses over it.
