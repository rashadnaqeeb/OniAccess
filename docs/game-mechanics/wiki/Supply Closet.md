# Supply Closet

In the
Supply Closet
you can find any blueprints you acquired, edit the default appearance of duplicants, make custom combinations of outfits or claim new available blueprints.
List of Blueprints
The full list of blueprints can be found in the
Gallery
Duplicant outfits
There are 3 option to customized outfits: Clothing, Overjoyed response and Atmo suit.
Clothing
Option to restyle a Dupe's outfit
The outfit of a duplicant can be customized in game after they are printed by selecting "Restyle" on their options.
A default outfit can also be set from supply closet. When the default outfit is changed, duplicants from the printing pod will appear with that new outfit but any already printed duplicants will have to be customized to it manually.
Overjoyed Response
The only overjoyed response that has a blueprint is Balloon Artist so only duplicants that have that overjoyed response show the option to change it.
Atmo Suit
Same as Clothing, it can be customized either per duplicant or as default for newly printed duplicants.
Wardrobe
In the wardrobe you can save any combination of outfits you want for quicker restyling of duplicants.
Claim Blueprints
Every week there are 6 blueprints available as
Timed drops
. Multiple copies of the same blueprint can be claimed. Most of the blueprints can be Recycled for Filament or be Printed. In order to obtain any blueprints,
Data Communication
option needs to be turned on.
