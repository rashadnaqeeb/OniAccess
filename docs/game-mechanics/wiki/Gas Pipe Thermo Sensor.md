Gas Pipe Thermo Sensor
is an automation sensor to detect the temperature of gas content in a
Gas Pipe
.
The thermo sensor has a setting for temperature and can be configured as either above or below this setting. It must be built in the same tile as a gas pipe segment. When the temperature of gas content in the pipe segment meets the configured criteria, the sensor will send a Green signal. Otherwise it will send a Red signal.
If the pipe segment is empty, the sensor retains the previous value.
