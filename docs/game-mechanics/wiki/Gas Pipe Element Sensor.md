# Gas Pipe Element Sensor

Gas Pipe Element Sensor
Sends a Green Signal when the selected Gas is detected within a pipe.
Element sensors can be used to detect the presence of a specific gas in a pipe.

Gas Pipe Element Sensor
is an automation sensor that detects the content of a
Gas Pipe
. It has an element filter for player to choose a type of Gas. It must be built in the same tile as a Gas Pipe segment. When the chosen Gas is detected in the pipe segment, it will output a Green signal. Otherwise it will output a Red signal.
If the gas pipe is empty, it outputs a Red signal.
Usage
Gas Pipe Element Sensor combined with
Gas Shutoff
can serve as alternative to
Gas Filter
, that consumes less power (10 W vs. 120 W). See
Guide/Loop Filters
.
