# Power Shutoff

Power Shutoff
Connects to an Automation grid to automatically turn Power on or off.
Does not affect circuitry preceding the switch.
Automated systems save power and time by removing the need for Duplicant input.

Power Shutoff
controls electricity flow using
Automation
signals. It disables underlying wire and does not affect other circuitry.
A Green automation signal allows electricity to flow. A Red automation signal prevents flow. Without other connections, this is toggled by the player.
Tips
It continues to function while entombed or submerged.
Power Transformers
and
Large Power Transformers
can be used in the same way, and are likely to be already placed in meaningful positions.
The Power Shutoff can be paired with an
Automation output building
(e.g. an
Atmo Sensor
) to allow it to turn on and off without player interaction.
Trivia
The activation/deactivation sound effect will produce different tones depending on the length of the automation wire. So it can be used
for music-boxes.
The pitch of the note will drop by 1 semitone per wire length increase, e.g 2 wire length (shortest possible) will produce a C6 tone and a 3 wire length will produce a B5.
