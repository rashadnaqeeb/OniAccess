A
Buried Object
is a hidden item within a solid
Resource
tile. The object is often a
seed
from a
plant
native to the
biome
it is located in. The object can be harvested by digging up the tile that contains it.
Any gravity affected blocks such as
sand
or
snow
upon landing on debris will create sand/snow with buried debris in it.
Note that buried
geysers
are not shown as buried objects this way. Instead, they can be recognized by the four tiles of
neutronium
under them.
