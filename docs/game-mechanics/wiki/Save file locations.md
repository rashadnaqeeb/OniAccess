Windows
- C:\Users\%USERNAME%\Documents\Klei\OxygenNotIncluded\save_files
Mac
- ~/Library/Application\ Support/unity.Klei.Oxygen\ Not\ Included/save_files
Linux
- ~/.config/unity3d/Klei/Oxygen Not Included/save_files
