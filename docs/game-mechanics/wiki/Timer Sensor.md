# Timer Sensor

Timer Sensor
Creates a timer to send
Green Signals
and
Red Signals
for specific amounts of time.
Timer Sensors create automation schedules for very short or very long periods of time.

Tips
The game engine appears to evaluates 5 ticks per second, so intervals are rounded to units of 0.2 s.
Bugs
Like all time-based sensors, it is very imprecise for large durations measurement at high cycle count.
