NOT Gate
is an
Automation
Building
that takes in an input signal and outputs the opposite signal. It is connected to other buildings via
Automation Wire
.
NOT Gate will effectively reverse the behavior of buildings or sensors that output automation signals but lack numerical settings, like the
Smart Storage
or
Gas Element Sensor
. It is usually unnecessary for sensors that detect numerical values like the
Atmo Sensor
since they can be configured to output either a Green or Red signal as desired.
