Starmap Location Sensor
can only be constructed in a rocket interior and signals when the rocket is at chosen locations. The sensor is associated with the
rocket
it is constructed within.
Location Filter
The sensor has a location filter to choose locations on the
starmap
which determine where the sensor will send a green signal. If the associated rocket is currently at a location chosen in the filter, then the sensor will send a green signal; otherwise, the sensor will send a red signal.
