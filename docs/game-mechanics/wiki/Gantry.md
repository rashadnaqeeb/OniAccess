Gantry
is a retractable scaffolding capable of extending into space occupied by
rocket
modules.  A gantry can be used to create a walk-able path for
Duplicants
to access various rocket modules such as the
Command Capsule
or
Sight-Seeing Module
, as well as the
Solid Fuel Thruster
or
Solid Oxidizer Tank
.  Gantries can also be used to create a walk-able path across multiple rockets.
In Spaced Out, gantries are no longer be damaged by rockets, even while extended.  If the non-retracting part of a gantry is built in the path of a rocket, launch will be prevented as the Launch Checklist reports that the launch path is not clear and landing is similarly prevented.
Tips
Although gantry consumes 1200 W of energy, it does so only for about 1.6 seconds while it is expanding or retracting. It is possible to use a single
Conductive Wire
for all of them by delaying their movements by 1.7 seconds via
BUFFER Gates
and
FILTER Gates
.
Jet Suits
can be used as an alternative to Gantry but is difficult to refuel inside the rocket.
Auto-Sweepers
can be used as an alternative for solid fuel and oxidizer insertion, although those might require heat-resistant materials.
