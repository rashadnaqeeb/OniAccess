# Thermo Sensor

Thermo Sensor
Sends a
Green Signal

Red Signal
when ambient Temperature enters the chosen range.
Thermo sensors can disable buildings when they approach dangerous temperatures.

at the same tile it is on to the set threshold. You can select whether it activates if it is above or below that threshold, meaning that, unlike most other sensors, it does not need a

If placed in a vacuum, it will read absolute zero. It will
not

Conduction Panel
