Conveyor Rail Germ Sensor
is an automation sensor to detect the germ count in a
Conveyor Rail
.
It has a count threshold and can be configured as either below or above this threshold.
It is useful for keeping contaminated debris separated from clean debris.
Sensor will perform while entombed or submerged.
