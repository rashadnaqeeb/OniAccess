# Mesh Tile

Mesh Tile
Used to build the walls and floors of rooms.
Does not obstruct Liquid or Gas flow.
Mesh tile can be used to make Duplicant pathways in areas where liquid flows.

Heat Exchange
For
heat exchange

the solid tile below it
