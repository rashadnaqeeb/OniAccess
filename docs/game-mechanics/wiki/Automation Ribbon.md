# Automation Ribbon

Automation Ribbon
A 4-Bit
Automation Wire
which can carry up to four automation signals.
Use a
Ribbon Writer
to output to multiple Bits, and a
Ribbon Reader
to input from multiple Bits.
Logic ribbons use significantly less space to carry multiple automation signals.

Automation Ribbon
is the same as 4
Automation Wires
compressed into one tile. It can be connected to other Automation buildings such as
Mechanized Airlocks
, and will only use the first
Bit
.
When
Automation Wires
are directly connected to an Automation Ribbon, it will cause overloading damage.
Since
Automation Wires
only carry 1 Bit, it is possible to use Automation Wires to extend an Automation Ribbon and recover 4 Bit data from Automation Wires.
To use the full 4 bits you will need to use a
Ribbon Writer
or a
Ribbon Reader
to get signal on to and out from the ribbon, respectively. Without these only the first bit is used. Since the materials required to use this is so much higher and it requires additional space, using
Automation Wires
is a great choice for smaller systems or earlier in the game.
