Releases trapped Pacu back into the world.
Can be used multiple times.
A fish release must be built above liquid to prevent released fish from suffocating.

Flooded

Liquid

Tips
If built under water, it will get a "Building Flooded" debuff and stop working. After some time it will be broken due to liquid pressure. So it must be built above the water surface.
