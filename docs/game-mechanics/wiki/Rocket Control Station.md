# Rocket Control Station

Rocket Control Station
Allows Duplicants to use pilot-operated rockets and control access to interior buildings.
Assigned Duplicants must have the
Rocket Piloting
skill.
Someone needs to be around to jiggle the controls when the screensaver comes on.

Rocket Control Stations
are buildings exclusive to
Rocket
interiors. Duplicants operating this building will apply a travel speed modifier depending on their piloting skill, overriding the autopilot modifier. They can be set to restrict access to certain buildings when the rocket is grounded or on certain starmap locations using the
Starmap Location Sensor
.
Each newly constructed spacefarer module comes with a control station pre-built inside.
A rocket cannot launch without a Rocket Control Station, and only one may be built per rocket.
