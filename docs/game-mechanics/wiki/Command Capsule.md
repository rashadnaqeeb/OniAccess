Command Capsule
Contains passenger seating for Duplicant Astronauts.
A Command Capsule must be the last module installed at the top of a rocket.
At least one astronaut must be assigned to the command module to pilot a rocket.

Automation
A Command Capsule has one automation output port and one automation input port. The output port indicates whether the rocket checklist is all completed and ready to take off. The input port instructs the rocket to launch.
The output port will continue outputting Green until the rocket leaves the map.
A basic automation solution is to connect the Command Capsule output port to a
FILTER Gate
, and then connect the FILTER Gate to the Command Capsule input port. The effect is that once a rocket is ready to take off, it will wait for X seconds before auto-launching. These X seconds are for
gantries
to retract and any unused fuels in fuel pipes to flow back to the storage. Note the gantries and fuel management needs additional automation solution.
Tips
Duplicants do not require any life-support while inside the Command Capsule and will not leave it without player's input, which allows to use it as a
suspended animation
chamber.
Sight-Seeing Module

Jet Suit

Gantry
