Stable
is a
room
that is required for
ranching
. It is created by placing a
Ranching Building
inside an enclosed room with a size of 12-96 tiles.
A stable's maximum
critter
capacity depends on each critter's individual tile space requirement.

If the
critter population requirement
is greater than the
room size
, all critters in the room will get the
Crowded
status.
Eggs
left in the room count towards the population requirement, but will give the
Cramped
status instead.
Morbs
do not count towards the number of max critters.
Shove Voles
will count towards the stable population, however, they do not care about any amount of overcrowding themselves.
Cuddle Pips
consider each egg type to only require 4 tiles of space.
Aquatic
critters
use the number of connected, non-navigable liquid cells for determining space requirements, including those inside Mesh Tiles and Pneumatic Doors. They do not need to be inside a Stable.
Tips
Critters might spread too much in maximum size stable, slowing down grooming, feeding and resource production. It can be mitigated with non-room breaking door.
It is inadvisable to keep open doors in ranches, even if critters can't run away, they still get stuck in open doorframes and will get debuffs as if entombed. It also interferes with buildings ability to count critters inside the room.
