Liquid Pipe Thermo Sensor
is an automation sensor to detect the temperature of liquid content in a
Liquid Pipe
.
The thermo sensor has a setting of temperature and can be configured as either above or below this setting. It must be built in the same tile as a liquid pipe segment. When the temperature of liquid content in the pipe segment meets the configured criteria, the sensor will send a Green signal. Otherwise it will send a Red signal. This sensor is mandatory when using a base water cooling system, as it makes preventing pipes from freezing much easier.
If the pipe segment is empty, the sensor retains the previous value.
