# Care Package

Care Packages
are only obtainable from the
Printing Pod
or the
Mini-Pod
. An active Printing Pod will have a chance to offer between one and two Care Package every three cycles.
If the Difficulty Option "Care Packages" was unchecked at world creation, no care packages will be offered by the Printing Pod.
Some items will not be offered as care packages until their discovery conditions are fulfilled (these can vary between Base Game and
Spaced Out!
).
All liquids arrive bottled. All critters and eggs spawn in a wild state at their default temperature (usually
20 °C
/
68 °F
) and might die shortly after spawning if the environment is inhospitable.
Available Care Packages
The base selection of care packages is dependent on the activation state of the
Spaced Out!
DLC.
If it is disabled, the care packages of the "Base Game" table are available, when it is enabled, the "Spaced Out!" table takes over.
Any additional active content pack DLC care package lists become available in addition.

Exploits
Saving and reloading with the Printing Pod ready will generate a new selection of blueprints or care packages. This can be exploited to shuffle the duplicants and care packages until the desired duplicant or care package is available.
Care packages can be
duplicated
when saving and reloading when both the item being delivered and the care package are spawned.
