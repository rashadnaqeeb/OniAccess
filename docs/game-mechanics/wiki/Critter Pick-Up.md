Critter Pick-Up
allows the automation of
critter
wrangling by
duplicants
.
When the total number of critters amongst those selected from the list exceeds the set maximum, an errand to wrangle one is created. This maximum value can be set to a value from 0 to 40.
A Critter Pick-Up is not required for the
Stable
and can be placed anywhere.
Tips
This building has a base overheat temperature of
75 °C
/
167 °F
and will overheat in
Oil Biome
if built with
Copper Ore
or
Iron Ore
. Choose
Gold Amalgam
as the building material instead.
Only critters selected in the building's critter list generate wrangle errands. Unselected critters do not.
Multiples of this building with different settings can control the numbers of critter variants, such as keeping a Stable with 1
Puft Prince
along with other Puft variants.
Placing a Critter Pick-Up in a room containing non-wranglable critters allows a Dupe to call those critters to the Pick-Up to be wrangled.
Trivia
All critters used to be wranglable in early phases of the game. Some of them were made non-wranglable before the official release because their wrangling caused frustrating pathing issues. The
Critter Drop-Off
retained the ability to wrangle all critters, with the Dupe visibly whistling critters so they come to the Critter Pick-Up, avoiding pathing issues. This was not considered a bug or an exploit. The ability to manually set a critter to be wrangled was added back later.
