Conveyor Rail Thermo Sensor
is an automation sensor to detect the temperature of debris content in a
Conveyor Rail
.
The thermo sensor has a setting of temperature and can be configured as either above or below this setting. It must be built in the same tile as a Conveyor Rail segment. When the temperature of debris content in the rail meets the configured criteria, the sensor will send a Green signal. Otherwise it will send a Red signal.
If the rail is empty, the sensor retains the previous value.
