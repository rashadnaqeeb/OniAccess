# Critter Sensor

Critter Sensor
Sends a
Green Signal
or a
Red Signal
based on the number of eggs and critters in a room.
Detecting critter populations can help adjust their automated feeding and care regiments.

Critter Sensor
compares the number of tamed eggs or
Critters
in a room with the set threshold. You are able to toggle between detecting either eggs or critters, or both. You can select whether it activates if it is above or below that threshold.
It does not count
Morbs
, because they cannot be tamed.
Tips
Critter sensors' count includes any critters or eggs in
Incubators
as well as any eggs in storage or on
Conveyor Rails
.
