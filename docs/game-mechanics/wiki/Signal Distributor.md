Signal Distributor
is a
Demultiplexer
Automation
Building
which can route one signal to one of four outputs. The signal is routed based upon two control inputs.
Signal Distributor
The Signal Distributor has three inputs.
Input Signal - represented with a circle around it
Most Significant Control (MSC) - directly opposite the input Signal and tied to the highest point of the triangle on the component
Least Significant Control (LSC) - next to the MSC and in the middle of the triangle on the component
The
Input Signal
will be sent to one and only one of the four output ports. The remaining output ports will be sent a
red
signal. The highest numbered output (3) is the one closest to the input signal. The outputs are in sequential order
with the lowest numbered output (0) furthest from the input signal and next to the LSC input.
