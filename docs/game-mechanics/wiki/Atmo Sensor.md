# Atmo Sensor

Atmo Sensor
Sends a
Green Signal
or a
Red Signal
when
Gas
pressure enters the chosen range.
Atmo Sensors can be used to prevent excess oxygen production and overpressurization.

Atmo Sensor
compares
gas
pressure at the same tile it is on to the set threshold. You can select whether it activates if it is above or below that threshold, meaning that, unlike most other sensors, it does not need a
NOT Gate
.
It must be surrounded by gas. Being placed in
vacuum
or liquid makes it read pressure as 0g.
Tips
Gas Pumps
will consume a constant 240 Watts regardless of how much gas they are pumping, so putting an Atmo Sensor set to "above 500 g" nearby will increase their power efficiency.
Atmo Sensors are helpful to prevent gas overpressure or to pump
geyser
-produced gas only when the geyser is active.
