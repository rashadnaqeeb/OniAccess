# Agriculture

Agriculture
is an
attribute of a duplicant
.
Effects
Trained
For each level of this attribute, the duplicant gets:
+5% farming speed
+2.5% tending speed
+3.3% seed chance when harvesting a plant
+10% farmer's touch duration
The duplicants can increase this attribute by:
doing the
farming
errand
, including
Micronutrient Fertilizer
creation in the
Farm Station
and uprooting plants.

Traits

Improved Farming I (+2)
Crop Tending (+2)
Improved Farming II (+2)

Early Bird (+2, but only in the
morning
)

)
