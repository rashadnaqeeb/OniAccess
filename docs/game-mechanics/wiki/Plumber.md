# Plumber

Plumbers
can execute
empty pipe
errand
, which extracts pipe's content of both types into transportable containers. This is a continuous task and it has to be cancelled manually as long as there is some content in the pipe.
This allows deconstructing pipes without spilling its contents into environment. It is less fitting for working with moving content in pipes, since contaminants has to be "caught" at the same tile the dupe works.
Plumbing can only be done by a duplicant with the Plumbing
skill
.
