# Equipment

This is a list of equipment
Duplicants
can wear in the game
Oxygen Not Included
. Exosuits and regular clothing can be combined and do not interfere with each other.
Most types of equipment are produced in the
Textile Loom
or the
Exosuit Forge
.
Using Exosuits
While basic clothing can be manually assigned once and forgotten, Exosuits require some additional infrastructure to be used continuously.
Sole Exosuits docks can be used only with manual management, since exosuits regularly run out of
Oxygen
(or
Petroleum
) it is not very practical. The same goes for non-powered and non-piped docks with checkpoints.
Checkpoints have to be connected to the row of docks that they serve. They prevent dupes from passing without exosuit equipped.
It is possible to add checkpoints on the both sides of the dock, and it will make duplicants wear Exosuits all the time, however it can cause path finding issues if there are no Exosuits available in the dock array. It is also wastes energy with unnecessary Exosuit refilling.
The dupes always store suit in the dock closest to the checkpoint. Thus
piping
should prioritize docks starting from the checkpoint side to minimize the downtime.
The dupes will unequip both types of Exosuits at checkpoint of any type, which might cause some troubles, so care should be taken.
It can be useful to put a
Gas
or
Liquid Valve
on input pipes to limit the buffers inside the docks.
