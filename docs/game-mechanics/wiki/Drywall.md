# Drywall

Drywall
Prevents gas and liquid loss in space.
Builds an insulating backwall behind buildings.
Drywall can be used in conjunction with tiles to build airtight rooms on the surface.

Drywall
protects a single tile against the
Space Exposure
effect of
Space Biome
, preventing the loss of liquids and gases.
It does not hide wires or pipes.
Drywalls can be built in the same tile as
Ladders
and
Fire Poles
, but not tiles or doors. They are a type of
Background Building
, which can be built behind most other buildings but not behind solid tiles.
Tips
While normally players aren't allowed to build
Mesh Tiles
or
Airflow Tiles
in the same space as a Dry Wall, attempting to do so would override one another. There is an exploit that can be utilized to still achieve this. By placing a ladder where a Dry Wall is, then queue a build order over the ladder, a duplicant will then only replace the ladder, leaving the Mesh/Airflow tile in place. This is helpful when trying to build certain things in
The Void
where you might want air/liquid flow between sections of a building without also exposing them to space.
