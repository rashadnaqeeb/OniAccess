# Storage Bin

Storage Bin
Stores the Solid resources of your choosing.
Resources left on the floor become "debris" and lower decor when not put away.

Resources do not exchange heat with the storage bin directly, but exchange heat with the environment, specifically the bottom cell of the storage bin, and the solid tile directly below the storage bin if it exists. This might cause resources inside the storage bin to phase change, such as
Ice
inside a Storage Bin will release
Water
to the environment when melted.
Usage
Storage bins provide a place to stock resources marked for sweeping, as duplicants will not sweep if there is no authorized storage compactor to store the resource.
They also act as time-savers by storing construction materials in an accessible place, or store consumable resources near their consumer (e.g. storing
Coal
near a
Coal Generator
).
Mechanics
A Storage bin placed in the world.
When destroying a storage bin, its contents will drop onto the ground.
Only the bottom square of the Storage Bin must be submerged in order to prevent resources from emitting gas while in storage.
Tips
Resources outside Storage Bins are still considered to be available, so there is no need to put resources into Storage Bins for the sake of doing it, especially when
Decor
is not important and there are other tasks for
Duplicants
to focus on.
You can use 1 category for each Storage Bin from start.
Raw Mineral
consists of the bulk of resources near the start, so it would need the largest number of Storage Bins to store. For the rest categories, 1-3 Storage Bins can be used for each, placed near where the corresponding resources will be consumed.
Storage bins emit bad decor, even if made from sandstone or granite. Placing them in rooms on the sides of the base is optimal, as they don't create much traffic.
Sweeping fills storage, increases decor, and decreases lag, but requires an unfilled, properly-configured storage bin. Use strong duplicants on sweep/delivery jobs, as they can carry more resources each turn.
By changing the priority of the storage bin, you can organize materials. So if you have a Storage Bin in which everything goes set to 1 and one only for
Bleach Stone
set to 5, the
Bleach Stone
will be put in the Storage Bin with highest priority. This can be handy for organizing resources such as
Coal
, seeds, etc.
Duplicants will also move items between storage bins based on priority, allowing you to transfer already stored items to new locations.
Automatic Dispenser
set to sweep only is a strong alternative to Storage Bins.
