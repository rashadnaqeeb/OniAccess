# Mailbox

Mailbox
Displays a single edible object
There's nothing quite like receiving home-made gifts in the mail.
Dimensions
2×2 tiles
Category
Story Trait
Decor
-15 (Radius: 3 tiles)
The
Mailbox
is a
Building
which can only be found inside the
Mysterious Hermit
story trait
point of interest, which cannot be built.
During the story trait it is used for a quest to display various food items. Afterwards it can't display items anymore.
When the story trait is completed the Mailbox can be deconstructed for 100 kg of
Steel
.
