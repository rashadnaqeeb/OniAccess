# Apothecary

Apothecary
Produces Medicine to cure most basic Diseases.
Duplicants must possess the Medicine Compounding Skill to fabricate medicines.
Duplicants will not fabricate items unless recipes are queued.
Some medications help prevent diseases, while others aim to alleviate existing illness.
