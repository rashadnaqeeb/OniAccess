# Red Alert

Red alert is a state that can be toggled on or off. Turning it on forces
Duplicants
to prioritize work over basic needs.
Description
To switch red alert on or off, click the exclamation point "!" next to the cycle time controls. When toggled on, the edges of the screen glow red and a notification highlighted red will appear. An alarm sound will also play.
While under red alert, all duplicants will enter an emergency state and ignore their needs (sleeping, bladder, etc.) to focus on work. This results in high amounts of
stress
from lacking sleep, full bladder, and hunger. This emergency state should only be used shortly, as using it for a long time may cause death from ignored needs.
Example Scenario
In certain scenarios, red alert will be needed for the
Colony
's survival. Red alert can be used to rescue a trapped duplicant during the
night
(while other duplicants are sleeping) and in imminent danger of death before daytime. In this scenario, the player can force all duplicants to excavate the trapped duplicant.
