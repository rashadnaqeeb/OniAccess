The
Orbital Cargo Module
is one of the the very first Modules with which to transport Material to other Planetoids. As opposed to other
Cargo Modules
, the orbital cargo module can send its contents down to a planet without the
Rocket
needing to land.
It has to be manually loaded, the Solid Rocket Port Loader will not load items into the Orbital Cargo Module.
Once in orbit over a destination, the module needs to be deployed from the star map. Upon deployment it will drop multiple
Interplanetary Payload
bullet-like packages over the destination's surface, 200 kg of payload per bullet. The payload can be unpacked manually or using the
Payload Opener
. Note that you cannot control where the payload bullets will land which means some of the cargo may be inaccessible to your dupes when they land on the planetoid. A
Targeting Beacon
can be used to circumvent this issue.
