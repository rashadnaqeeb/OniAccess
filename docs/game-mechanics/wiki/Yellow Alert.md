# Yellow Alert

Yellow Alert is the highest priority a task can be given. This priority level overrides
duplicant
specific priorities.
Description
Yellow Alert is set by clicking on the red double "!!" to the right of the task priory bar. This does not override basic duplicant needs like
red alert
, but makes it the highest priority task for every available duplicant. Any duplicant with the skill can be given the task. When applied to a task with no definitive end such as auto-bottle on a
bottle emptier
, the task will remain in a state of yellow alert until cancelled or deprioritized by the player.
Example Scenario
Some tasks may be very important and need to be done immediately. This may happen when a wall fails holding back water, a door needs to be built for a ranch, or an exposed
sporechid
is emitting
zombie spores
.
