# Rust Biome

The
Rust Biome
is a cold
Biome
filled with
Rust
and
Iron Ore
. It also contains
Nosh Sprouts
and
Dasha Saltvines
; which are both exclusive to this biome.
Strategy
While the
Rust Biome
is cold; the temperature is high enough to not be much of a problem (unless you are on a cold map). All gases present here sink below oxygen, making airflow from oxygenated parts of the map far easier to control. The
Rust
found here can be used to generate
Oxygen
and
Iron Ore
with a
Rust Deoxidizer
. If you are just after the
Iron
though, melting rust is a more efficient conversion.
Because
Dasha Saltvines
are exclusive to this biome, if the
Cluster
does not contain one it is impossible to get the GMO A-OK Achievement, unless the
Printing Pod
offers a seed.

Rust Biome
The orange-brown oasis of the Rust Biome is home to many unusual flora and fauna. It contains the resources for several intermediate technologies.
When combined with the
Rust Deoxidizer
,
Rust
can produce many of a colony's basic needs.
The Squeaky Puft, a frequent resident of the Rust biome, is a renewable source of
Bleachstone
and can be domesticated for such purposes.
Dreckos
can also sometimes be found in these biomes, which are a great source of
Phosphorite
and fibre for making
Textile Production
.
Two plants found in this biome, the
Nosh Bean
and the
Dasha Saltvine
, can both produce food that will add significant
Morale
value for my Duplicants.
