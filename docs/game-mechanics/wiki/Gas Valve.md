control the amount of gas flowing through a
Gas Pipe
. Best usage after branching, or stopping gas without losing contents before deconstructing. Because it allows gas flow only one way, it can also be used to give the game a hint what flow direction the gas pipe is having, when the pipe grid is getting more complicated.
Flow
Gas Valve has a flow setting, ranging from 0 g/s to 1000 g/s. This setting is the upper limit of the output gas flow. For example, if the setting is set to 500 g/s and input flow is 1000 g/s, the output flow is limited to 500 g/s and excess gas is blocked. If the setting is set to 500 g/s and input flow is 300 g/s, the output flow is 300 g/s.
A valid flow setting is rounded the first decimal place and has the following limits:
the minimum flow is 0 g/s;
the maximum flow is 1000 g/s;
the minimum, nonzero flow is 0.1 g/s (100 mg/s).
Switching the flow setting of Gas Valve requires Duplicant
operation
, and takes some time.
Tips
Broken valves do not affect building operation. A broken valve from overheat or meteor damage still functions exactly the same as a non-broken valve.
