AND Gate
is a basic logic gates introduced in the
Automation Upgrade
. An AND Gate will output Green if and only if
both
of its inputs are receiving Green. These particular gates are useful in scenarios where you want, say, a device to turn on when a sensor becomes active
and
a switch is turned on.
The two nodes with the arrow facing
toward
the device are the inputs, while the third node is the output.
